1. Description:  
This project is a simple HTML5 webpage styled using CSS3.  
It demonstrates basic use of semantic tags, external stylesheets, image usage, and accessibility best practices.  

The page includes:  
- A heading and intro text  
- Two content sections  
- An image with an alt tag  
- A styled layout with colors, spacing, and fonts  

--------------------------------------------------

- styles.css
